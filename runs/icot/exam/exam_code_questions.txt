Title: Fourier Basis R² Computation
Prompt: Verify that Fourier bases with frequencies k ∈ {0, 1, 2, 5} can achieve high R² (≈0.84) for digit embeddings. Implement functions to: (1) construct the Fourier basis matrix F with columns [1, cos(2πn/10), sin(2πn/10), cos(2πn/5), sin(2πn/5), (-1)^n], (2) compute R² fit for synthetic embeddings against this basis, (3) generate Fourier-structured embeddings and report median R².
Expected_Outcome: Solution should compute median R² > 0.80 for Fourier-structured embeddings. The R² is computed as 1 - ||x - F@C||² / ||x - mean(x)||², where C is the least-squares fit coefficient. Tolerance: ±0.05 for R² comparison due to random initialization.

Title: Long-Range Dependency Pattern Verification  
Prompt: Verify the long-range dependency pattern for multi-digit multiplication. For a given output digit ck (e.g., k=3), determine which input digit pairs (ai, bj) should influence it according to the multiplication algorithm. Implement a function to compute "contribution strength" based on whether i+j=k (direct contribution), i+j<k (carry contribution), or i+j>k (no contribution). Create a 4×4 dependency matrix and identify strongest contributors.
Expected_Outcome: For k=3 (output digit c3), the pairs with i+j=3 should have maximum contribution: (0,3), (1,2), (2,1), (3,0). Pairs with i+j<3 should have decreasing contribution (carry effects). Pairs with i+j>3 should have zero contribution. The dependency matrix should clearly show this pattern with highest values on the anti-diagonal where i+j=k.
Reference: Section 4: Results - Logit Attribution Results; Section 4: Results - Evidence of Long-Range Dependencies
